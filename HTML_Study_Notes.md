# HTML

## 1. 웹과 HTML

### 1.1 웹
- **인터넷**: 전 세계 컴퓨터가 연결된 거대한 네트워크
- **웹(World Wide Web)**: 인터넷 위에서 동작하는 서비스
    - **동작 원리**
        - **요청(Request)**: 브라우저가 서버에 웹 페이지를 요청
        - **처리**: 서버가 요청을 처리하고 웹 페이지를 준비
        - **응답(Response)**: 서버가 파일들(HTML, CSS, JS 등)을 브라우저에 전송
        - **렌더링(Rendering)**: 브라우저가 받은 파일들을 해석해서 웹 페이지로 변환

### 1.2 HTML
- **HTML(HyperText Markup Language)**: 웹페이지의 구조와 내용을 정의하는 마크업 언어
    - **마크업 언어**: 컴퓨터에게 '이 부분은 제목이야', '이 부분은 문단이야'라고 알려주는 표시 언어
    - **특징**
        - **구조화된 문서**
        - **태그 기반**: <태그명> </태그명> 구조
        - **하이퍼링크**: 다른 웹 페이지로 연결하는 링크 생성 가능
        - **플랫폼 독립적**: 어떤 기기에서든 동일하게 표시

### 1.3 HTML 문서의 기본 구조
- **<!DOCTYPE html>**: HTML 문서임을 선언
- **<html>**: 모든 내용을 감싸는 최상위 요소
- **<head>**: 사용자에게는 보이지 않지만 중요한 정보들이 들어가는 곳
    - **<title>**: 웹 브라우저 탭에 표시되는 제목
- **<body>**: 사용자가 실제로 보게 될 내용들이 들어가는 곳

### 1.4 개발자 도구
- **개발자 도구로 할 수 있는 일들**
    - HTML 구조 확인: 웹페이지가 어떤 태그들로 구성되어 있는지
    - CSS 스타일 분석: 각 요소에 어떤 디자인이 적용되었는지
    - 실시간 수정: 코드를 바로 수정해서 결과를 즉시 확인
    - 오류 찾기: JavaScript 에러나 문제점 발견
    - 성능 분석: 페이지 로딩 속도나 최적화 상태 점검
- **개발자 도구 단축키**
    - Windows/Linux: F12
    - Mac: Cmd + Option + I
    - 마우스 우클릭: "검사" 또는 "요소 검사" 선택

### 1.5 태그와 속성의 기본 구조
- **태그 기본 구조**: <태그명 속성="값"> 내용 </태그명>
    - 여는 태그: <태그명>
    - 닫는 태그: </태그명>
    - 속성: 태그에 추가 정보 제공
    - 내용: 태그 사이에 들어가는 실제 콘텐츠

### 1.6 태그
- **<h1> ~ <h6>**: 제목 태그
- **<ul>, <li>**: 순서 없는 목록
- **<strong>**: 굵은 글씨
- **<em>**: 기울임 글씨
- **<p>**: 문단
- **<a>**: 링크

### 1.7 속성
- **href**: 연결할 주소
- **alt**: 이미지를 불러올 수 없을 때 표시되는 대체 텍스트
