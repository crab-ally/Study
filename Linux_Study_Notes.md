# Linux

## 1. Ubuntu와 터미널 감 잡기

### 1.1 터미널(Terminal)
- 마우스로 클릭해서 작업하던 것을 글자로 직접 명령하는 창

### 1.2 기본 명령어
- **pwd(Print Working Directory)**: 현재 작업중인 디렉토리의 경로 출력
    - **기본형**: pwd
- **ls(List)**: 현재 디렉토리의 파일 및 폴더 목록 출력
    - **기본형**: ls
    - **옵션**
        - **ls -l**: 자세히 보기
        - **ls -a**: 숨김 파일 포함
- **cd(Change Directory)**: 디렉토리 이동
    - **기본형**: cd 경로
        - **cd ..**: 상위 디렉토리로 이동
        - **cd ~ (또는 cd)**: 홈 디렉토리로 이동
        - **cd /**: 루트 디렉토리로 이동

### 1.3 경로
- **절대 경로**: 루트 디렉토리(/)부터 시작하는 경로
- **상대 경로**: 현재 디렉토리부터 시작하는 경로

### 1.4 주의할 점
- **대소문자 구분**: Linux는 대소문자를 구분함
- **공백**: 파일명이나 디렉토리명에 공백이 있으면 " "로 감싸줘야 함

## 2. 파일, 디렉토리 다루기

### 2.1 기본 명령어
- **mkdir(Make Directory)**: 디렉토리 생성
    - **기본형**: mkdir 폴더이름
    - **옵션**
        - **mkdir -p**: 계층구조의 상위 디렉토리까지 한 번에 생성
- **touch**: 빈 파일 생성
    - **기본형**: touch 파일이름
- **cp(Copy)**: 파일 복사
    - **기본형**: cp 원본 대상
    - **옵션**
        - **cp -r**: 디렉토리 복사
- **mv(Move)**: 파일 이동, 파일 이름 변경
    - **기본형**
        - **이동**: mv 파일 폴더
        - **이름 변경**: mv 원본 새이름
- **rm(Remove)**: 파일 삭제
    - **기본형**: rm 파일이름
        - **주의**: 복구 불가능, 확인 없이 삭제됨, 휴지통 없음
    - **옵션**
        - **rm -r**: 디렉토리 삭제
        - **rm -f**: 강제 삭제

## 3. 파일 내용 보기 + 편집

### 3.1 기본 명령어
- **cat(Concatenate)**: 쉘에 파일 전체 내용 출력
    - **기본형**: cat 파일이름
    - **주의점**: 파일이 길면 화면이 한 번에 지나감
- **less**: 별도 쉘에서 파일 전체 내용 출력
    - **기본형**: less 파일이름
    - **사용법**
        - q: 종료
        - 방향키: 스크롤
        - b: 이전 페이지
        - space: 다음 페이지
- **head**: 파일 내용 앞부분 출력
    - **기본형**: head 파일이름 (기본 앞 10줄 출력)
    - **옵션**
        - **head -n 숫자**: 줄수지정
- **tail**: 파일 내용 뒷부분 출력
    - **기본형**: tail 파일이름 (기본 뒷 10줄 출력)
    - **옵션**
        - **tail -n 숫자**: 줄수지정
        - **tail -f**: 파일 내용 실시간 출력
            - Ctrl + C: 종료
- **nano**: 파일 내용 편집
    - **기본형**: nano 파일이름
    - **사용법**
        - Ctrl + O: 저장
        - Enter: 확인
        - Ctrl + X: 종료

## 4. 검색과 찾기

### 4.1 기본 명령어
- **find**: 파일 검색
    - **기본형**: find 경로 -name "파일이름"
- **grep**: 텍스트 검색
    - **기본형**: grep "검색어" 파일이름
    - **옵션**
        - **grep -i**: 대소문자 구분 없이 검색
        - **grep -r "검색어" 디렉토리**: 디렉토리 전체 검색
        - **grep -n**: 줄 번호 표시
- **which**: 실행 파일 경로 찾기
    - **기본형**: which 명령어
- **find + grep**: 특정 디렉토리에서 특정 텍스트 검색
    - **기본형**: find 경로 -name "*.py" | xargs grep "검색어"

## 5. 권한과 실행

### 5.1 권한
- **첫 글자**: 파일(-), 폴더(d)
- **권한 구조**: [소유자][그룹][기타]
    - **r(read)**: 읽기
    - **w(write)**: 쓰기
    - **x(execute)**: 실행

### 5.2 기본 명령어
- **chmod(Change Mode)**: 파일 권한 변경
    - **기본형**: chmod 권한 파일이름
        - chmod +x 파일이름
            - 모든 사용자에게 실행 권한 부여
        - chmod 숫자 파일이름
            - r(4), w(2), x(1)
- **sudo(Super User Do)**: 관리자 권한으로 실행
    - **기본형**: sudo 명령어

## 6. 패키지 설치와 업데이트

### 6.1 개념
- **저장소**: 프로그램 목록이 저장된 서버
- **패키지**: 설치 가능한 프로그램 단위

### 6.2 기본 명령어
- **apt(Advanced Package Tool)**: 패키지 관리자
    - **기본형**: apt 명령어
    - **apt update**: 최신 패키지 목록 가져오기
    - **apt upgrade**: 설치된 패키지 최신 버전으로 업그레이드
    - **apt install 패키지이름**: 패키지 설치
        - apt update 필수
    - **apt remove 패키지이름**: 패키지 삭제
        - **sudo apt purge 패키지이름**: 패키지 삭제 + 설정 파일 삭제
    - **apt search 키워드**: 패키지 검색
    - **apt list --installed**: 설치된 패키지 목록 보기
        - **apt list --installed | grep 패키지이름**: 특정 패키지 확인

## 7. 프로세스 & 시스템 확인

### 7.1 개념
- **프로세스**: 실행 중인 프로그램
- **PID**: 프로세스 ID

### 7.2 프로세스 기본 명령어
- **ps(process status)**: 현재 터미널에서 실행 중인 프로세스
    - **기본형**: ps
    - **옵션**
        - **ps aux**: 시스템 전체 프로세스 보기
- **top**: 실시간 프로세스 모니터링
    - **기본형**: top
    - **사용법**
        - q: 종료
- **kill**: 프로세스 종료
    - **기본형**: kill PID
    - **옵션**
        - **kill -9 PID**: 강제 종료
            - 정상 종료 안 될 때 사용

### 7.3 시스템 기본 명령어
- **uname**: 현재 os 정보 출력
    - **기본형**: uname
    - **옵션**
        - **uname -a**: 시스템 전체 정보 출력
- **df(disk free)**: 디스크 사용량 확인
    - **기본형**: df
    - **옵션**
        - **df -h**: 사람이 읽기 쉬운 형태로 출력
- **free**: 메모리 사용량 확인
    - **기본형**: free
    - **옵션**
        - **free -h**: 사람이 읽기 쉬운 형태로 출력
