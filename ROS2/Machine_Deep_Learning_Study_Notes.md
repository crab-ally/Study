# Machine Learning & Deep Learning

---

## 1. 기본 개념

### 라벨 (Label)

정답 데이터

예시:

| 입력 데이터 | 라벨 |
|------------|------|
| 고양이 사진 | 고양이 |
| 강아지 사진 | 강아지 |

---

### 모델 (Model)

입력을 받아 출력을 만드는 함수

```text
입력 데이터 → 모델 → 예측 결과
```

---

### 선형 모델 (Linear Model)

직선으로 표현할 수 있는 관계

```text
y = wx + b
```

각 변수의 의미:

- `x` : 입력값(Input)
- `w` : 가중치(Weight)
- `b` : 편향(Bias) → 모델의 출력값을 위아래로 이동시키는 보정값
- `y` : 예측값(Output)

> w가 클수록 입력 변화에 민감하게 반응

---

## 2. 머신러닝

입력 데이터(x)와 출력(y)의 관계를 학습하여 적절한 가중치(w)와 편향(b)을 찾는 과정

### 2-1. 지도학습 (Supervised Learning)

입력 데이터와 정답 라벨을 함께 사용하여 학습하는 방법

```text
입력 데이터 + 정답 라벨
          ↓
        학습
          ↓
       예측 모델
```

예시:

- 고양이/강아지 분류
- 집값 예측
- 스팸 메일 분류

---

### 2-2. 비지도학습 (Unsupervised Learning)

정답 라벨 없이 데이터의 구조나 패턴을 찾는 방법입니다.

```text
입력 데이터만 제공
        ↓
   패턴 탐색
```

예시:

- 고객 그룹 분류
- 데이터 군집화(Clustering)
- 차원 축소(Dimensionality Reduction)
